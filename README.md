# GARCIA STARTER PACK

Esqueleto de API completa para iniciar novos projetos.


# Rodando

após instalar as dependências, suba a infra com o comando

```
npm run infra
```

e depois suba a api com

```
npm run start
```


Para desligar os containers do Docker antes e após o uso, rode:

```
npm run stop:all
```
